### 👨‍💻 Sobre Mim
Sou apaixonado por tecnologia, autodidata e focado em inovação. Atualmente trabalho no Itaú Unibanco com load balancers F5 BIG-IP, desenvolvendo automações para otimizar processos e reduzir custos.

### 💼 Experiência
**Estagiário de TI - Itaú Unibanco (2024)**
- Implementação de automação de processos na área de telecomunicações, visando otimizar fluxos de trabalho.
- **IUAUTOMATIONBALANCER**: Automação integrada entre backend e frontend para otimizar a geração de scripts de gerenciamento de load balancers, reduzindo o tempo de execução de 2 horas para 2 minutos.
- **Coleta Backup F5OS**: Automação em Python para a coleta de backups dos BIG-IP F5 e transferência automática para servidor SFTP, garantindo a centralização e segurança dos dados.
